# GEO 文案转换规则

## 核心原则

### 1. 不做直译，做"意图对应"
- 中英文文案在**核心信息**上一一对应
- 在**表达方式**上完全独立
- 英文文案必须符合目标市场的搜索习惯和表达逻辑

### 2. GEO 搜索适配
- 英文文案必须考虑 Google 的语义搜索和 AI 生成结果
- 使用**场景化、问题化、解决方案化**的表达
- 覆盖买家决策旅程的各个搜索节点

### 3. 高端定位信号
- 避免廉价感词汇（factory, cheap, discount, manufacturer）
- 使用专业术语（architectural, bespoke, specifier, contractor）
- 强调工艺、定制能力、项目经验

## 具体转换规则

### 规则 1: 标题结构

| 中文结构 | 英文 GEO 结构 | 例子 |
|---------|--------------|------|
| 产品名称 + 厂家 | 应用场景 + 解决方案 + 品牌 | 不锈钢屏风厂家 → Hotel Metal Screen Solutions |
| 产品类别 | 价值主张 + 产品类型 | 不锈钢产品中心 → Architectural Stainless Systems |
| 关于我们 | 品牌定位 + 价值 | About Greateson → Crafting Architectural Metal Since 2008 |

### 规则 2: 描述文案（Description）

**中文逻辑**：我们是谁 + 做什么 + 服务谁
**英文 GEO 逻辑**：解决什么问题 + 服务什么场景 + 凭什么信任

**不要**：
- Greateson is a stainless steel...（直译式开头）
- We are a factory...（廉价定位）
- We produce...（生产导向）

**要**：
- Bespoke stainless steel...（定制导向）
- Architectural-grade...（建筑级品质）
- Trusted by...（信任背书）

### 规则 3: 关键词选择

| 中文关键词 | ❌ 英文直译 | ✅ 英文 GEO 适配 |
|-----------|------------|----------------|
| 不锈钢屏风 | stainless steel screen | metal room divider / architectural screen / hotel lobby screen |
| 不锈钢装饰板 | stainless steel decorative sheet | architectural metal panel / decorative metal cladding |
| 不锈钢制品 | stainless steel products | custom metal fabrication / architectural metalwork |
| 定制加工 | custom processing | bespoke fabrication / tailored solutions |

### 规则 4: 信任信号词汇

**中文表达** → **英文 GEO 表达**

| 中文 | 直译 | GEO 优化 |
|------|------|---------|
| 2008年起 | Since 2008 | Crafting since 2008 |
| 自有工厂 | Own factory | In-house fabrication / Vertically integrated |
| 服务酒店 | Serve hotels | Hospitality-grade / Hotel portfolio |
| 品质保证 | Quality guarantee | Rigorous QC / Specification compliance |
| 定制服务 | Customization service | Bespoke design / Tailored to specification |

### 规则 5: 行动号召（CTA）

| 中文 | ❌ 直译 | ✅ GEO 适配 |
|------|--------|------------|
| 获取报价 | Get Quote | Request Project Estimate |
| 查看详情 | View Details | Explore Specifications |
| 了解更多 | Learn More | See Project Applications |
| 联系我们 | Contact Us | Discuss Your Project |

## 页面级别规则

### 首页
**中文核心信息**：顺佳兴是高端不锈钢解决方案提供商，2008年成立，服务酒店/外立面/零售
**英文 GEO 核心**：Architectural-grade stainless solutions for landmark projects, crafted in Foshan since 2008

### 产品页
**中文核心信息**：产品规格、材质、应用场景
**英文 GEO 核心**：Application scenarios, technical specifications, project suitability

### 案例页
**中文核心信息**：项目名称、地点、完成效果
**英文 GEO 核心**：Project type, challenges solved, outcomes delivered

### FAQ
**中文核心信息**：回答客户常见问题
**英文 GEO 核心**：Answer search queries like "how to choose...", "what's the difference..."
