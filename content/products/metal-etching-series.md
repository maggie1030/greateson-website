###核心介绍
金属蚀刻板是一种通过化学腐蚀工艺，在不锈钢表面形成图案、纹理或立体效果的装饰材料。
通过在金属表面覆盖保护层，仅对指定区域进行腐蚀，使图案“刻”入板材内部，从而形成具有层次感和精细细节的装饰效果。
蚀刻工艺通常不会单独使用，而是与拉丝、喷砂、电镀、镀铜、做旧、转印等多种表面处理工艺结合，实现更丰富的视觉表现。
其特点是图案精细、层次丰富、可高度定制，广泛应用于电梯装饰、墙面设计、商业空间及高端定制项目。
###颜色和表面处理
蚀刻本身是“纹理工艺”，颜色通常来自后续复合工艺：
1. 常见组合颜色：钛金 / 香槟金 / 玫瑰金；黑钛 / 灰钢；古铜 / 青铜 / 红铜；做旧复古色系；多色填色蚀刻
2. 可实现效果：
镀色蚀刻（整体有颜色）；退钛蚀刻（蚀刻部分无色或不同色）；多色填充蚀刻（图案填色）
👉 颜色与蚀刻顺序有关（先蚀刻 or 先镀色效果不同）
###工艺优势
- 可实现高精度复杂图案（如山水画、人物图案）
- 表面具有立体感与层次感
- 支持个性化定制（图纸→模具→生产）
- 可与多种工艺叠加（电镀 / 做旧 / 转印等）
- 适用于大面积装饰与高端项目
- 视觉效果高级，适合品牌空间打造
- 可实现连续拼接图案（需高精度工艺）
👉 感光蚀刻可实现1mm级精细线条（高端项目核心优势）
###技术参数
- 基材材质：304（大面积推荐） / 201 / 316
- 最大长度：约6米
- 常规厚度：0.5mm - 3.0mm
- 蚀刻深度：
  - 3-5c：浅纹理（镜面）
  - 8c以上：有手感
  - 10-15c：明显立体感
  - 特殊可达40-50c
- 工艺方式：
  - 感光蚀刻（高精度）
  - 丝印蚀刻（高效率）
- 模具费用：
  - 感光蚀刻：约150元/㎡
  - 丝印蚀刻：约1200元/套
👉 大面积蚀刻建议使用304，否则容易出现杂质、水印等问题
###应用场景
电梯轿厢板、背景墙、展示柜、橱柜、吊顶、商业空间装饰、门头及装饰边条
👉 电梯板是蚀刻板最典型应用场景之一
###工艺流程
1. 基础蚀刻流程
图案设计与制模 → 表面涂覆保护油墨→ 自然烘干→ 化学蚀刻（控制时间决定深度）→ 脱模清洗→ 表面处理
2. 常见复合工艺流程
电镀蚀刻：开板 → 拉丝/8K → 喷砂 → 蚀刻 → 电镀 → 过油 → 贴膜
镀铜蚀刻：开板 → 蚀刻 → 磨砂 → 镀铜 → 发黑 → 拉丝 → 过油
做旧蚀刻：蚀刻 → 镀铜 → 氧化 → 手工纹理 → 抛光 → 喷色 → 封油
转印蚀刻：蚀刻 → 转印（木纹/石纹）→ 烘干 → 贴膜
###FAQ
Q1: 蚀刻板可以做多精细的图案？
A: 使用感光蚀刻可以实现1mm级细节，适用于高精度图案。
Q2: 感光蚀刻和丝印蚀刻有什么区别？
A:
- 感光蚀刻：精度高，适合小批量和复杂图案
- 丝印蚀刻：成本低，适合大批量生产，但精度略低
Q3: 蚀刻板为什么会掉色或镀不上色？
A: 主要原因是蚀刻后清洗不干净，残留药水会影响后续电镀效果。
Q4: 大面积蚀刻为什么建议用304？
A: 因为201杂质较多，蚀刻后容易出现水印、黑点等问题。
Q5: 蚀刻板可以拼接吗？
A: 可以，但高精度拼接必须使用感光蚀刻，否则会出现误差。
Q6: 蚀刻深度怎么选择？
A:
- 装饰：3-5c
- 有手感：8c+
- 立体效果：10-15c
Q7: 生产周期多久？
A: 常规7-10天，复杂工艺（如做旧、多色）约10-15天。
